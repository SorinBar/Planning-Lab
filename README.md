# Planning-Lab
Dependencies:
  * Python3

For Linux OS:
  * Set Up:
    * chmod +x Planning-Lab.sh
    * chmod +x setup.sh
    * ./setup

  * Run:
    * ./Planning-Lab.sh

For Non-Linux OS:
  * Set Up: 
    * [Install python3-pip]
    * pip3 install asyncio
    * pip3 install websockets
  * Run:
    * python3 server.py
    * [Open index.html in Browser]
